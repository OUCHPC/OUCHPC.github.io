# OUCHPC.github.io

中国海洋大学HPC社团的官方网站。

## 简介

这是一个使用纯HTML和CSS构建的静态网站，用于展示社团信息、新闻动态和招募信息。

## 如何本地预览

1. 克隆本仓库:

   ```bash
   git clone https://github.com/OUCHPC/OUCHPC.github.io.git
   ```

2. 在浏览器中直接打开 `index.html` 文件即可。

## 部署

该网站会自动部署到GitHub Pages。任何推送到 `main` 分支的更改都会触发自动更新。

访问网站：[https://ouchpc.github.io/](https://ouchpc.github.io/)
